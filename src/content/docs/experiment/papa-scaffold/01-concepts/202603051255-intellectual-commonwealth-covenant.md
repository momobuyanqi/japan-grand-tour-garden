# 智识共同体契约：父女互喻式数字生存协议

> **“在数字荒原中，我们不建立层级，我们建立并联。”**

## 一、 核心哲学：分布式主权与互喻教育

1.  **独立 OS 原则**：父亲（WYGG）与女儿（momo）各自拥有独立的“人生 OS”。双方互不依附，但在“日本壮游”这一特定的智识场域中建立**对等桥接**。
2.  **互喻式学习（Mutual Metaphor）**：父亲通过搭建“学习脚手架”提供资源供养；女儿通过活跃的田野反馈与生命体验赋予资源新的灵性。这种双向流动的过程即为“互喻”。
3.  **社会父亲的智识馈赠**：资源共享并非一种管教工具，而是一种基于“实证逻辑”的智识馈赠，旨在缩短女儿在现实迷雾中的摸索成本。

## 二、 责权利边界

1.  **父亲的义务**：
    *   作为“社会父亲”，承诺将调研得到的最高密度的文哲资料、田野路径及技术方案实时同步至共享节点。
    *   负责维护共享仓库的底层稳定与历史安全（补救机制）。
2.  **女儿的权利**：
    *   拥有共享仓库的**写操作主权**。她可以自由在其中建立自己的“田野笔记”、“影像流”或“任务计划”。
    *   有权质疑、修正或重构父亲提供的任何调研设计。
3.  **共同的契约**：
    *   **诚实提交**：每一次代码推送（Push）都应承载真实的生命思考。
    *   **历史敬畏**：尊重每一份文档的演进痕迹，不轻易通过暴力覆盖破坏他人的智识产出。

## 三、 数字伦理：错误即资产

我们认可在协作中可能出现的“误删”、“分支污染”或“逻辑冲突”。在本契约下，这些意外不被视为“失误”，而被视为**“数字素养的试炼”**。我们将通过 Git 的历史溯源机制共同学习如何“回滚”与“修复”，在错误中习得分布式协作的真谛。
