# 操作手册：东京摄影田野 3 日典型行程建议

## 🗓️ DAY 1：体制内的凝视与致敬
- **上午**：访问东京工艺大学或日本大学艺术学部（视开放情况），感受日本摄影教育的校园气息。
- **下午**：东京摄影美术馆 (TOP Museum)。重点解读经典大师展。
- **晚上**：茑屋书店 (代官山或银座)。观察大众对顶级影像画册的消费场景。

## 🗓️ DAY 2：新宿二丁目的“半地下”乌托邦
- **下午**：新宿古书店巡游。重点挖掘绝版摄影杂志。
- **傍晚**：**Place M** (新宿二丁目)。看展并尝试参与工作坊或点评。
- **深夜**：观察新宿巷弄（森山大道的街道）。尝试用手机或相机捕捉那种“粗粒子、失焦、黑白”的东京瞬间。

## 🗓️ DAY 3：书店、画廊与视觉消费
- **上午**：写真集食堂 Megutama (恵比寿)。在摄影画册的包围中进行关于“居场所”的讨论。
- **下午**：POST、SUPER LABO STORE 等先锋摄影/艺术书店。
- **傍晚**：银座 Morioka Shoten。观察“一间书店、一本书”的极简主义商业美学。
