# 智识共享脱敏协议：隐私保护与反泄密行为规范

## 🛡️ 核心目标

由于本项目库已通过 GitHub 共享，尽管是私有仓库，但仍面临协作者（如 momo）在不知情的情况下通过屏幕截图、转发文案或社交媒体发布而导致的信息泄露风险。本协议旨在建立一套“自动脱敏”习惯。

## 🚫 严禁直接录入的敏感信息 (Blacklist)

1.  **私人即时通讯工具**：严禁录入任何人的私人微信号、手机号。
2.  **非公开的办公/居住细节**：严禁录入受访者详细到卧室、不公开实验室或私人住宅内部路径的描述。
3.  **小学侣真实姓名全称**：在公开发布的文档中，一律使用“代号”或“ initials”（如：A 同学、L 小侣）。
4.  **未授权的内部草稿**：带有“CONFIDENTIAL”或“INTERNAL ONLY”标记的研究草案。

## 🔄 脱敏替代方案 (Standard Masks)

-   **手机号/微信**：统一替换为 `[REDACTED: PRIVATE CONTACT]` 或指向其官方联络表单。
-   **私人住址**：模糊为区域级（如：`〒160-XXXX 新宿区某处`）。
-   **涉及法律/财务的敏感词**：使用代称或缩写。

## ⚖️ 协作伦理：转发前置确认

**momo 同步须知**：
1.  **禁止截屏转发**：严禁将本仓库内的原始 Markdown 内容直接截屏发至朋友圈或社交媒体。
2.  **引语限制**：若需引用其中的观点，请务必先将文案复制到本地，进行人工脱敏（去除地址、邮箱、具体姓名等）后再发布。
3.  **同步警告**：如果您不确定某项内容是否可以对外分享，请默认其为“绝密”，并向父亲（WYGG）咨询。
