# 社会化学习空间在日本 (Socialized Learning Space in Japan)

**建立日期**：2026-03
**项目性质**：针对辍学、弃学少年的跨国社会化学习实验。

## 📂 项目结构与逻辑 (Project Architecture)

本项目采用 **Antigravity OS** 标准的五维架构，确保从理念到实操的闭环：

1. **`01-concepts/` (设计逻辑与契约)**
   - **含义**：项目的“灵魂”与“地基”。
   - **用途**：存放愿景声明、伦理准则、社会父亲角色定义以及“责权利声明书”等顶层哲学文档。
   - **核心文件**：
     - `202603042304-Japan-Great-Tour-Vision.md` (壮游愿景)
     - `202603042310-Covenant-of-Rights-and-Responsibilities.md` (责权利契约)
     - `202603050138-日本壮游目的地设计.md` (目的地设计思考原稿)
     - `202603051057-field-missions-standard-protocol.md` (田野调研标准化执行协议)
     - `202603051255-intellectual-commonwealth-covenant.md` (父女智识共同体契约)
     - `202603051310-data-desensitization-and-privacy-protocol.md` (智识共享脱敏协议)

2. **`02-destinations/` (目的地研究)**
   - **含义**：智识地标的“物理地图”。
   - **用途**：对摄影书店、博物馆、文豪足迹、社会干预机构等具体地理坐标的背景调研。

3. **`03-field-missions/` (田野调研任务)**
   - **含义**：交给小学侣的“行动脚本”。
   - **用途**：将目的地研究转化为具体的 PB（项目式学习）调研课题，如“一根萝卜的革命”、“居场所扫描”等。

4. **`04-itinerary/` (行程规划)**
   - **含义**：项目的“时间脊椎”。
   - **用途**：每日的具体行动动线、交通逻辑与落地安排。

5. **`05-toolkits/` (策略与工具箱)**
   - **含义**：实战的“仪表盘”。
   - **用途**：存放专家联系名单、驻留申请方案、文化签证策略及调研伦理操作手册。

## 🎯 核心命题
在高度成熟的资本主义都市圈（关东/关西）寻找教育主权。通过摄影、建筑、农业等具体抓手，为小学侣搭建智识与生存的“脚手架”。
