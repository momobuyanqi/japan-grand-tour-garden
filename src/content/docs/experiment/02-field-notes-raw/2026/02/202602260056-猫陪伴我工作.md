## 🖋️ 今日学习内容

### 远程治理
- 用 Discord 远程指挥 OpenClaw，在手机端就能触发全量同步或内容回撤。哪怕人在田野，也能实时掌控发布状态。

### 对象存储
- 完成 Cloudflare R2 的初始化，单独建桶，作为网站的云端分发网关。顺手解决了 GitHub 存储限制，还把全球访问速度拉满。

### 双轨影像归档
- 建立 Master / Proxy 的双轨制：Master 高清原件留在本地，作为长期资产；Proxy 用于分享与发布，在传播与主权之间保持克制和平衡。

### 自动化管线
- 用 Python 写了一整套自动流程：照片进入系统后，自动完成转码 → UID 命名 → 双轨处理 → R2 上传。人退出，系统继续工作。

### 数字民族志实验室
- 在数字花园里开了 Digital Ethnography Lab，记录这些技术基建如何一步步生成。
- 这不只是技术笔记，而是对「记录本身」的民族志观察。
