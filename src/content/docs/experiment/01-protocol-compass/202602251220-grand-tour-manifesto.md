# 🌏 扶桑乘槎录：一种社会替代教育的旅行人类学实验

> “乘槎而行，博考当世。这不仅是跨越海洋的航行，更是对人类文明高度的深度透视。”

## 🕯️ 历史的薪火：致敬先行者
本计划之名“乘槎”，旨在致敬清末先行者**斌椿（Binchun）**与**张德彝（Zhang Deyi）**。他们在 1866 年跨越重洋，留下《乘槎笔记》与《航海述奇》，完成了中国近代最早的全球观察。

今日，momo 与小学侣们再次“乘槎”东渡。不同于百年前对工业化的初步窥探，我们此次是带着**社会学、人类学的显微镜**，去剖析一个已经进入“高成熟期”的社会样本。

## 🔬 学术坐标：米德、本尼迪克特与今和次郎
我们将这场壮游定锚在以下坐标系中：
1. **人类学冒险**：效仿**玛格丽特·米德**的参与式调查，深入日本社会的褶皱——从辍学少年到尼特族，从社区养老到 CSJ 农业。
2. **国民性镜像**：带着**本尼迪克特《菊与刀》**的冷峻，去观察当下日本在 ACG 文化与传统仪式、现代建筑与历史遗迹之间的复杂缠绕。
3. **考现学实证**：以**今和次郎**的“考现学”为方法，将东京、大阪、京都作为活体实验室，记录“当下的历史”。

## 🌱 媒介实验：数字花园里的 Learning in Public
通过 `astro-digital-garden`，我们将这次壮游转化为一次**“媒体化科考”**：
- **实时采样**：田野现场的即时捕获。
- **跨时空互喻**：将日本样本与之前的洛阳、北京调研进行对比。
- **公开构建**：展示一种“以世界为教室”的社会替代教育方案。

---
**📅 壮游航线**：东京 → 大阪 → 奈良 → 京都
**🔭 研究重点**：城市考现、尼特族生存样态、ACG 空间人类学、私人博览建筑、中日关系民间观察。
